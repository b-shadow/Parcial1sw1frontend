# Frontend

Frontend Angular standalone para la plataforma de tramites.

## Stack base

- Angular 21 standalone
- Tailwind CSS
- PrimeNG
- PrimeIcons
- Formly con tema PrimeNG
- JointJS
- Apache ECharts
- ngx-echarts

## Ejecutar localmente

```bash
npm install
npm run start
```

## Convenciones

- UI base con PrimeNG.
- Ajustes de layout y espaciado con Tailwind.
- Formularios dinamicos con Formly.
- Diagramas renderizados con JointJS a partir de JSON; no como canvas libre en la primera fase.
- Dashboards con ECharts.
